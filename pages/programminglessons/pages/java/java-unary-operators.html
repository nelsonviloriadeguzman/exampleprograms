Assignment Operator assigns the value on its right to the operand on its left. This operator can also be used on objects to assign object references(will discuss later).

Code: 

  
  public class Main {
    public static void main(String[] args) {

      String fullName = "John Doe";
      int age = "18";

    }
  }

Concept Description
= Assignment (or equal) symbol is used to assign the value of "John Doe" to the fullName variable and the value of 18 to the age variable.
