if-then-else statement provides a last path of execution when an "if" clause evaluates to false.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      boolean isPassed = false;

      if (isPassed == true){
        System.out.println("Passed!");
      } else {
        System.out.println("Failed!");
      }

    }
  }

Output: 

        
  Failed!

Concept Description
else is the last alternative block of code that executes when none of the preceding if or else if conditions are true.
