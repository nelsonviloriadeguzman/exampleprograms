Comments is used to add explanatory notes to the code.

Code: 

  
  public class Main {
    public static void main(String[] args) {
      // This is a single-line of comments.
      System.out.println("Hello World!");
    }
  }

  public class Main {
    public static void main(String[] args) {
      /* 
          These are
                  multi-line
                            comments
      */
      System.out.println("Hello World!");
    }
  }

Concept Description
// double slash is a symbol used to write single-line comments. It starts with //
/* */ slash, asterisk is a symbol used to write multi-line comments. It starts with /* and ends with */
