if-then statement tells your program to execute a certain section of code only if a particular test evaluates to true.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      boolean isPassed = true;

      if (isPassed == true){
        System.out.println("Passed!");
      }

    }
  }

Output: 

        
  Passed!

Concept Description
if keyword tests the condition. It executes the if block if the condition is true.
() hold the condition that decides whether the code inside the block should run.
isPassed is used to compare for equality.
== operator used to compare two values for equality.
true is a keyword used to compare for equalit.
{ marks the beginning of a block of code.
} marks the end of a block of code.
