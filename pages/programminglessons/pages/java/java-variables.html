Variables are containers for storing data values, like numbers and characters.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      String thisIsTextStorage = "Hello World!";
      System.out.println(thisIsTextStorage);

      int thisIsNumberStorage = 7;
      System.out.println(thisIsNumberStorage);

    }
  }

Output: 

        
  Hello World!
  7

Concept Description
String String is a class, a non-primitive data type used to store, create and manipulate sequence of characters which represented as object.
thisIsTextStorage thisIsTextStorage is the name of a string variable.
= Assignment operator is used to assign a data to a variable.
"Hello World!" a text, string, value putted / inserted to thisIsTextStorage variable.
int int is a primitive data type used to store, create and manipulate number
from -2147483648 to 2147483647.
thisIsNumberStorage thisIsNumberStorage is the name of an int variable.
7 a number, int, value putted / inserted to thisIsNumberStorage variable.
; semi colon is used to end a statement.
