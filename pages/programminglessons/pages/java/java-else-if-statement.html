else if statement provides a many path of execution when an "if" clause evaluates to false.

Code: 

  
  public class Main {
    public static void main(String[] args) {

        int grade = 60;

        if (grade >= 96) {
          System.out.println("Outstanding");
        } else if (grade >= 86) {
          System.out.println("Very Satisfactory");
        } else if (grade >= 76) {
          System.out.println("Satisfactory");
        } else if (grade >= 60) {
          System.out.println("Fairly Satisfactory");
        } else {
          System.out.println("Did Not Meet Expectations");
        }
        
    }
  }

Output: 

        
  Failed!

Concept Description
else if alternative block of code between if and else that executes when none of the preceding if or else if conditions are true.
