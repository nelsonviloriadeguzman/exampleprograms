Syntax specifies the required combination and sequence of characters making up correctly structured code.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      System.out.println("Hello World!");
      System.out.println(7);

    }
  }

Output: 

        
  Hello World!
  7

Concept Description
System System is one of the core classes in Java.
. dot notation is use to access / use / call the properties such as variables or methods of an object or a class.
out out is a reference of PrintStream class inside System class.
println println is a method inside out reference to print a text.
() opening and closing parentheses indicate the execution of a function or method.
"Hello World!" a text, string, value or argument passed into the method to be printed. The characters must be between two double quotation marks.
7 a number, int, value or argument passed into the method to be printed. Double quotation marks are not needed.
; semi colon is used to end a statement.
