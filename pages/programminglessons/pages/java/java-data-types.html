Data Types refers to the type of value a variable can store / hold.

Primitive data types includes byte, short, int, long, float, double, boolean and char.

Non-primitive data types includes String, Arrays and Classes (will discuss later).

Code: 

  
  public class Main {
    public static void main(String[] args) {

    // byte can store value from -128 to 127
    byte byteVariable = 127;
    
    // short data type can store value from -32,768 to 32,767
    short shortVariable = 32767;
    
    // int data type can store value from -2,147,483,648 to 2,147,483,648
    int intVariable = 2147483647; 

    // long data type can store value from -9,223,372,036,854,775,808 to -9,223,372,036,854,775,807
    long longVariable = 9223372036854775806L;
    
    // float data type can store value from 6 to 7 decimal places.
    float floatVariable = 3.14f;
    
    // double data type can store value up to 15 decimal places.
    double doubleVariable = 3.14159;
    
    // boolean data type has only two possible values: true and false.
    boolean booleanVariable = true;

    // char data type can store a single character.
    char charVariable = 'D';

    System.out.println(byteVariable);       
    System.out.println(shortVariable);     
    System.out.println(intVariable);         
    System.out.println(longVariable);       
    System.out.println(floatVariable);     
    System.out.println(doubleVariable);   
    System.out.println(booleanVariable); 
    System.out.println(charVariable);       

    }
  }

Output: 

        
    127
    32767
    2147483647
    9223372036854775806L
    3.14f
    3.14159
    true
    D
