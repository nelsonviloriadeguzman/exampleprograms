Identifiers are sequences of characters in code that identify variables, functions, methods, classes, etc. In most languages, identifiers are case-sensitive and not enclosed in quotation marks.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      int age = 18

    }
  }

Concept Description
age age is an identifier used to name the integer variable.
Main Main is an identifier used to name a class.
main main is an identifier used to name a method within the Main class.
String[] String[] is a Predefined class name. Predefined means that It's already defined and provided by Java.
args args is an identifier used to name a string variable.
