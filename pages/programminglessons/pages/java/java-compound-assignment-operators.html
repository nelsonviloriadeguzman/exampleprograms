Compound Assignment Operators are shortcuts that do a math operation and assignment in one step.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      double givenNumber = 10;

      givenNumber += 1;                // 10 + 1
      System.out.println(givenNumber); // prints 11.0
  
      givenNumber -= 2;                // 11 - 2
      System.out.println(givenNumber); // prints 9.0
  
      givenNumber *= 3;                // 9 * 3
      System.out.println(givenNumber); // prints 27.0
  
      givenNumber /= 4;                // 27 / 4
      System.out.println(givenNumber); // prints 6.75
  
      givenNumber %= 5;                // 6.75 % 5
      System.out.println(givenNumber); // prints 1.75

    }
  }

Output: 

  
  11.0
  9.0
  27.0
  6.75
  1.75

Concept Description
+= += was used to add givenNumber by 1
-= -= was used to subtract givenNumber by 2
*= *= was used to multiply givenNumber by 3
/= *= was used to divide givenNumber by 4
%= %= was used to divide givenNumber by 5 and get the remainder
