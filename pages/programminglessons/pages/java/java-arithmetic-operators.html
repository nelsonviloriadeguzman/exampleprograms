Arithmetic Operators perform addition, subtraction, multiplication, division and modulus.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      int sumResult = 1 + 1;
      int differenceResult = 2 - 2;
      int productResult = 3 * 3;
      int quotientResult = 4 / 4;
      int remainderResult = 5 % 5;

      System.out.println("Sum is "+ sumResult);
      System.out.println("Difference is "+ differenceResult);
      System.out.println("Product is " + productResult);
      System.out.println("Quotient is " + quotientResult);
      System.out.println("Remainder is " + remainderResult);

    }
  }

Output: 

  
  Sum is 2
  Difference is 0
  Product is 9
  Quotient is 1
  Remainder is 0

Concept Description
+ Addition operator was used for 1 + 1.
- Subtraction operator was used for 2 - 2.
* Multiplication operator was used for 3 * 3.
/ Division operator was used for 4 / 4.
% Modulus operator was used for 5 % 5.

You can also use operators with variables.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      int input1 = 20;
      int input2 = 10;

      int sumResult = input1 + input2;
      int differenceResult = input1 - input2;
      int productResult = input1 * input2;
      int quotientResult = input1 / input2;
      int remainderResult = input1 % input2;

      System.out.println("Sum is "+ sumResult);
      System.out.println("Difference is "+ differenceResult);
      System.out.println("Product is " + productResult);
      System.out.println("Quotient is " + quotientResult);
      System.out.println("Remainder is " + remainderResult);

    }
  }

Output: 

    
  Sum is 30
  Difference is 10
  Product is 200
  Quotient is 2
  Remainder is 0

Concept Description
input1 input1 is the first variable used for operators.
input2 input2 is the second variable used for operators.

You can also use operators with variables that have already been assigned.

Code: 

  
  public class Main {
    public static void main(String[] args) {

      int input1 = 20;
      int input2 = 10;

      int sumResult = input1 + input2;
      int differenceResult = sumResult - input2;
      int productResult = differenceResult * input2;
      int quotientResult = productResult / input2;
      int remainderResult = quotientResult % input2;

      System.out.println("Sum is "+ sumResult);
      System.out.println("Difference is "+ differenceResult);
      System.out.println("Product is " + productResult);
      System.out.println("Quotient is " + quotientResult);
      System.out.println("Remainder is " + remainderResult);

    }
  }

Output: 

    
  Sum is 30
  Difference is 20
  Product is 200
  Quotient is 20
  Remainder is 0

Concept Description
sumResult sumResult was used for Difference Result.
differenceResult differenceResult was used for Product Result.
productResult productResult was used for Quotient Result.
quotientResult quotientResult was used for remainderResult Result.
